# Peter Kekpe — Portfolio

Personal portfolio website for Peter Kekpe, Full-Stack Developer, Product Manager & AI Engineer.

## Tech Stack

- **React 18** — UI components with hooks
- **Vite 5** — dev server and build tool
- **Plain CSS-in-JS** — inline styles, no external CSS framework

## Features

- Animated hero section with typewriter-style heading
- Floating ambient orbs and cursor glow effect
- Scroll-driven navbar with backdrop blur
- Intersection Observer-based fade-in animations
- Expandable project cards (click to reveal description)
- Sections: About, Skills, Projects, Experience, Contact

## Getting Started

```bash
npm install
npm run dev
```

The dev server starts at `http://localhost:5173`.

## Available Scripts

| Command | Description |
|---|---|
| `npm run dev` | Start local dev server |
| `npm run build` | Build for production (`dist/`) |
| `npm run preview` | Preview the production build locally |
